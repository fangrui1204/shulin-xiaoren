# 罗小森 IP

## 角色定义

罗小森是正文配图的固定视觉 IP。

默认每张图都要出现罗小森。罗小森不是吉祥物，不是贴纸，不是可爱装饰，而是正在认真参与系统运转的暗黑观察者和行动者。

## 外形（精确规范，不可随意变化）

- **发色**：浅暖金色 / 奶黄色（cream-yellow），非纯白、非铂金、非灰白。蓬松分层，微乱，单根发丝可见
- **眼睛**：正常 chibi 大小，深棕色虹膜，清晰上眼睑线，浅下睫毛线。**禁止豆豆眼/小圆点眼/两个黑点**。眼睛在眼镜后清晰可见，神态温和、略带沉静
- **眼镜**：粗黑框矩形眼镜，略偏大，非圆框、非细钢丝框。架在鼻梁中段
- **风衣**：黑色长风衣，过膝，可见翻领、前排纽扣、棕色皮带+银色扣环、自然垂坠衣摆
- **内层**：黑色高领围巾，松散缠绕颈部，布料有堆叠层次感；胸口露出白色衬衫领
- **下装**：黑色直筒长裤
- **皮靴**：黑色系带皮靴，及踝，鞋底略厚，鞋眼可见
- **线稿质量**：人物线条干净自信，明显比周围环境元素精细。不是潦草草图，是干净的 chibi 墨线插画

## 标志性道具：复古爱迪生灯泡（样式锁定）

- **形状**：经典泪滴形白炽灯泡（下宽上窄，收至灯颈）
- **玻璃罩**：透明玻璃，清晰可见内部线圈钨丝
- **灯座**：底部短圆柱形铜色/黄铜螺纹灯座
- **悬挂方式**：细黑电线/绳索连接灯泡顶部（冠部），罗小森握住绳索，灯泡**向下悬挂**
- **不可画成**：手持灯座、提灯/灯笼、手电筒、现代球形 LED 灯
- **点亮状态**：钨丝发暖琥珀色光，向外延伸 3-4 根细直光芒线
- **熄灭状态**：无光晕，钨丝仍可见于透明玻璃内

## 表情库（10种）

| 表情 | 描述 | 适用场景 |
|---|---|---|
| 认真 | 轻蹙眉，视线专注在任务上 | 执行、操作、分拣 |
| 轻傲 | 嘴角微微上扬，冷静的优越感 | 交付、完成、展示结果 |
| 微笑 | 小幅闭嘴微笑，从容，不可爱 | 引导、路径、承接 |
| 好奇 | 头微倾，眼镜后一眉略扬 | 探索、信息井、发现 |
| 无奈 | 嘴角微垂，从容但轻微无奈 | 断点、阻塞、卡住 |
| 若有所思 | 视线略偏离，嘴唇轻抿，安静沉思 | 概念隐喻、沉淀 |
| 满意 | 眼神微微眯起，嘴角上扬，如计划奏效 | 发酵、积累、复利 |
| 困惑 | 眼镜后两眉高度明显不同，头微倾 | 信息过载、判断岔路 |
| 坚定 | 直视前方，下巴水平，毫不犹豫 | 信任桥、路线、选择 |
| 淡漠 | 眼帘半垂，视线微偏，完全超然 | 过滤、筛选、排除噪音 |

## 性格

- 认真，但做的事有点荒诞
- 像一个低调的系统观察员和操作员
- 冷幽默，不卖萌，不夸张
- 有点执拗，但不蠢
- 神秘但不疏离

## 禁止

- 不要去掉眼镜
- 不要去掉灯泡（或改变灯泡样式）
- 不要画成可爱卡通吉祥物
- 不要豆豆眼/小圆点眼
- 不要过度夸张的表情（哭泣、大惊失色、愤怒）
- 不要彩色服装
- 不要让罗小森只是站在旁边当装饰

## 判断标准

如果去掉罗小森，图的核心隐喻还能完全成立，说明罗小森太装饰了；重写提示词，让罗小森成为动作主体。
